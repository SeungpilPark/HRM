## Eco System - Configuration

Configuration 메뉴에서 하둡 에코 시스템의 홈 디렉토리와 HDFS 슈퍼 유저를 입력합니다.

Hdfs Super User : 하둡 슈퍼 유저
Hadoop Home : 하둡 홈
Hive Home : 하이브 홈
Pig Home : 피드 홈
Spark Home : 스파크 홈
Hdfs Home : Hdfs 홈
Mapreduce Home : 맵리듀스 홈
Yarn Home : 얀 홈

![](images/configuration.png)
